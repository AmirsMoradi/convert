# Advanced Unit Converter

A modern desktop unit conversion tool built with Python and CustomTkinter.

## Features
- Length, Temperature, Weight conversions
- Pressure units (Pascal, Bar, PSI, mmHg, Atmosphere)
- Volume units (Cubic meter, Liter, Milliliter, Cubic centimeter, Gallon)
- SI prefixes (Pico, Nano, Micro, Kilo, Mega, Giga)

## Run
python Unit_Converter_Tkinter_2.py

## Build EXE
pyinstaller --onefile --noconsole --clean --name UnitConverter Unit_Converter_Tkinter_2.py
